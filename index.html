2. 通配符选择器

3. Id选择器

以#号开头，在当前页面中带有ID属性值的设置样式，一个页面中的ID属性值不能重复。

4. 类选择器

以 . 符号开头的 . 标签可以包含多个类选择器，在class标签中用空格隔开，显示效果会同时出现

5. 复合选择器：

有两个或多个基础选择器通过不同方式组合而成的

  • 标签指定选择器（既….又…..）
    又称为交集选择器，由两个选择器构成，其中第一个为标记选择器，第二个为class或id选择器，两个选择器之间无空格
  • 后代选择器（包函选择器）
    用来选择元素或元素组的后代，写法是把外层标记写在前面，内层写在后面，中间空格分隔，当标记发生嵌套时，内层成为外层的后代
  • 并集选择器
    是各个选择器通过逗号连接而成的，任何选择器都可以作为并集选择器的一部分，如果定义的样式完全相同或部分相同，就可以利用这选择器定义相同样式
  • 子元素选择器
    是让CSS选择器只能选择儿子辈的元素，以 > 号命名
    6. 属性选择器
    格式：标签+【属性】+【属性值】
    Input [type=”text”]

(1).定义flexbox伸缩盒

语法：

“Flex-flow”方向

基本上，Flex项目是沿着main axis(从main-start向main-end)或者cross axis(从cross-start向cross-end)排列。

  • main axis:Flex容器的主轴主要用来配置Flex项目。注意，它不一定是水平，这主要取决于flex-direction属性。
  • main-start | main-end:Flex项目的配置从容器的主轴起点边开始，往主轴终点边结束。
  • main size:Flex项目的在主轴方向的宽度或高度就是项目的主轴长度，Flex项目的主轴长度属性是width或height属性，由哪一个对着主轴方向决定。
  • cross axis:与主轴垂直的轴称作侧轴，是侧轴方向的延伸。
  • cross-start | cross-end:伸缩行的-
    配置从容器的侧轴起点边开始，往侧轴终点边结束。
  • cross size:Flex项目的在侧轴方向的宽度或高度就是项目的侧轴长度，Flex项目的侧轴长度属性是width或height属性，由哪一个对着侧轴方向决定。

2.定义伸缩方向 flex-direction

取值：

  • row：主轴与行内轴方向作为默认的书写模式。即横向从左到右排列（左对齐）。
  • row-reverse：对齐方式与row相反。
  • column：主轴与块轴方向作为默认的书写模式。即纵向从上往下排列（顶对齐）。
  • column-reverse：对齐方式与column相反。

transform语法

  • transform: none|transform-functions;
  • transform 属性的初始值是none，适用于块元素和行内元素
  • transform-functions;设置变换函数

函数值：

  • translate() 移动元素对象，即基于X轴和Y轴重新定位元素的位置
  • rotate() 元素顺时针旋转给定的角度。允许负值，元素将逆时针旋转
  • scale() 元素的尺寸会增加或减少，根据给定的宽度（X 轴）和高度（Y 轴）参数
  • skew() 元素翻转给定的角度，根据给定的水平线（X 轴）和垂直线（Y 轴）参数
  • matrix() 定义矩阵变换，即基于X轴和Y轴重新定位元素的位置
  1. 定义旋转

十二、设计3D变换

CSS3的3D变换主要的函数：

  • 3D位移：包括translateZ（）和translate3d（）函数
  • 3D旋转：包括rotateX（）、rotateY（）、rotateZ（）和rotate3d（）函数
  • 3D缩放：包括scaleZ（）、scale3d（）函数
  • 3D矩阵：包括matrix3d函数

1.定义位移

主要包括包括translateZ（）和translate3d（）这两种函数。translate3d（）是元素在三维空间移动，语法如下：

Translate3d （tx,ty,tz,）

  • tx : 代表横向坐标位移向量的长度
  • ty : 代表纵向坐标位移向量的长度
  • tz ：代表Z轴位移向量的长度，单位不能取百分比

2.定义3D缩放

主要有scaleZ（）、scale3d（）函数，当scale3d 的XY轴同时为1，那么效果等同于scaleZ（sz）当元素大于1时，元素放大，反之小于1大于0.01时，元素缩小，语法如下：scale3d （sx，sy，sz）

  • sx：横向缩放比例
  • sy: 纵向缩放比例
  • sz: Z轴缩放比例
  • s: 制定元素每个点在Z轴上的比例 scaleZ（s）
  • scaleZ（-1）定义一个原点在Z轴的对称点
  • scaleZ（）、scale3d（）函数单独使用时没有任何效果

3.定义旋转

  • rotateX（）、rotateY（）、rotateZ（）和rotate3d（）函数：
  • rotateX（）：指定一个元素围绕X轴旋转，旋转的量被定义为指定的角度
  • rotateY（）：指定一个元素围绕Y轴旋转，旋转的量被定义为指定的角度
  • rotateZ（）：指定一个元素围绕Z轴旋转，旋转的量被定义为指定的角度
    反之如果值为负数就是逆时针旋转
    a是旋转角度
  • rotate3d（），旋转由一个【x,y,z】向量并经过元素原点定义
  • rotate3d（x,y,z,a）

x: 是一个0~1之间的数值，主要用来描述元素围绕X轴旋转的矢量值

y: 是一个0~1之间的数值，主要用来描述元素围绕Y轴旋转的矢量值

z: 是一个0~1之间的数值，主要用来描述元素围绕Z轴旋转的矢量值

a: 是一个角度值，主要用来指定元素在3D空间旋转的角度，正值为顺时间旋转，反之负值为逆时针旋转

十三、设计过渡动画

1.设置过渡属性

transition-property 属性用来定义过渡动画的CSS属性名称，语法如下：

  • transition-property：none | all | [ ] [ ‘，’ ]*；
  • none：表示没有元素
  • all：默认值，表示针对所有元素，包括before个after伪元素
  • IDENT：指定的CSS属性列表。几乎所有色彩，大小或位置等相关属性

2．设置过度时间

  • transition-during 用来定义转换动画的时间长度，语法如下
  • transition-during： [, ]*;
  • 初始值为0，适用于所有元素，以及before和after

3.设置延迟时间

  • transition-delay用来定义开启过渡动画的延迟时间，语法如下：
  • transition-delay： [, ]*;

4.设置过渡动画类型

transition-timing-function 用来定义过渡动画的类型

属性值：

  • ease：初始值，平滑过渡
  • linear：线性滑过
  • ease-in：由慢到快
  • ease-out：由快到慢
  • cubic-bezier:特殊的立方贝塞尔曲线效果

5.设置触发方式

动态伪类 作用元素 说明
: link 只有链接 未访问的链接
: visited 只有链接 访问过的链接
: hover 所有元素 鼠标经过元素
: active 所有元素 鼠标点击元素
: focus 所有可被选中的元素 元素被选中

JavaScript事件包括click，focus，mouse move，mouse over，mouseout等

十四、设置帧动画

1.设置关键帧CSS使用@keyframes定义关键帧

参数值：

  • animationname 定义动画的名称
  • Keyframes-selecto 定义帧的时间未知，就是动画时长的百分比
  • Css-styles 表示一个或多个合法的CSS样式属性

2.设置动画属性

（1）定义动画名称animation-name

（2）定义动画时间animation-duration

（3）定义动画类型 animation-timing-function
（4）定义延时时间 animation-delay
表示开始执行一段一段时间后才被应用；如果设置时间为0就代表是默认时间

（5）定义播放次数 animation-iteration-count

属性值：

  • 默认值为1，动画从开头到播放开始到结束一次
  • Infinite 表示无限次；
  • 如果取值为非整数，导致动画结束一个周期的一部分
  • 如果取值为负值，则导致在交替周期内反向播放动画

（6）定义播放方向 animation-direction

属性值：

  • normal：默认值，动画的每次循环都向前播放
  • Alternate：表示第偶数次向前播放，第奇数向相反向播放

（7）定义播放状态 animation-play-state
属性值：

  • running：正在播放
  • Paused: 已暂停

（8）定义播放外状态 animation-fill-mode
属性值：

  • none 不设置动画对象之外的状态
  • Forwards: 设置对象状态为动画结束时候的状态
  • Backwards：设置对象状态为动画开始时候的状态
  • Both：设置对象状态为动画开始或结束时候的状态
